Proveedor

Pasos para conectar JSONBin.io (gratis):

  1. Abre jsonbin.io en otra pestaña
  2. Crea una cuenta gratuita (o inicia sesión)
  3. Haz clic en tu perfil → API Keys
  4. Copia tu Secret Key (empieza con $2a$)
  5. Pégala abajo y toca Conectar

⚠️ La app debe abrirse en un servidor web (no como archivo local). Usa las instrucciones de la sección "Alojar en la web" abajo.

Pasos para conectar Firebase (Google):

  1. Ve a console.firebase.google.com
  2. Crear proyecto → nombre libre → continuar
  3. Lado izquierdo: Realtime Database → Crear base de datos
  4. Elige la región más cercana → modo de prueba (30 días)
  5. Copia la URL (ej: https://mi-app-default-rtdb.firebaseio.com)
  6. En Reglas pon: ".read":true,".write":true

Los datos se guardan solo en este dispositivo.

Alojar la app en la web (necesario para nube)

Para que la app funcione desde cualquier dispositivo y se conecte a la nube, necesitas subirla a un servidor. Las opciones más fáciles y gratuitas:

Opción A — Netlify (más fácil, recomendado):
  1. Ve a netlify.com → Crea cuenta gratis con Google/GitHub
  2. En el dashboard: arrastra y suelta el archivo alpha_app_v3.html al área de deploy
  3. ¡Listo! Te da una URL como https://xxx.netlify.app
  4. Comparte esa URL con todos los dispositivos → todos ven los mismos datos
Opción B — GitHub Pages:
  1. Crea cuenta en github.com
  2. Nuevo repositorio → sube el archivo HTML → activa GitHub Pages en Settings
  3. URL: https://tu-usuario.github.io/repo/alpha_app_v3.html
